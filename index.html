@lists

> Unordered list

The list which don't have serial no's. We can specify the type of bullets like disc, square, etc.

Example

  • int
  • char
  • float
  • boolean

> Unordered inline list

The list which don't have serial no's as well as the items of list are in single line or row.

Example

  • C
  • C++
  • Java
  • Python
  • java script

> Ordered list

The list which have serial no.s as bullets.

Example

  1. Facebook
  2. Amazon
  3. Apple
  4. Netfix
  5. Google

> Reverse ordered list

The list which have serial no's but in reverse order

Example

  1. Milk
  2. Bread
  3. Butter
  4. Eggs
